## Циклы

В java есть несколько видов циклов, познакомлю вам с ними постепенно. Циклы используются для повторяющегося выполнения некой операции пока истинно некоторое условие.
Каждый повтор операции называется итерацией, то есть если цикл 10 раз напишет слово "Привет", то это
значит, что он выполнил 10 итераций.

### Цикл while
Работает по принципу "пока условие истинно, выполнять операцию".
Например, пока счётчик не будет равен 10, прибавлять к нему 1:

```java
    int i = 0;

    while (i < 10) {
        ++i;
    }

    System.out.println("Counter: " + counter);
```

### Цикл for
Более навороченная версия цикла `while`, которая позволяет делать более короткую
запись. Сигнатура у него следующая for (начальное состояние; условие; изменение состояния на каждой итерации) {}
Пример из секции выше через цикл `for`:

```bash
    for (int i = 0; i < 10; ++i) {
    }
```

Думаю, что сходство с прeдыдущим примером очень сильное. Задаём `i` (int i = 0)
таким же образом, уловие такое же (i < 10), действия в цикле такие же (++i), 
просто всё сконцентрировано в одном месте.

У нас получился цикл с пустым телом, который толком ничего не делает, кроме 
увеличения `i` на 1. При этом после завершения цикла, мы не можем получить значение
переменной `i`, так как она жила только внутри цикла. Для наглядности добавим
вывод значения `i` на каждой итерации и после самого цикла:

```bash
    for (int i = 0; i < 10; ++i) {
        System.out.println("Inner i: " + i);
    }
    System.out.println("Outer i: " + i);
```

При компиляции вы получите ошибку о том, что невозможно найти переменную с именем
`i` во втором принте. Смело его удаляйте и запускайте снова.
